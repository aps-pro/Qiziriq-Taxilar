## Description
This project is a simple static website for **Qiziriq Turbo Taxi 1420**. It provides essential information about the taxi service, including contact details, services offered, and an easy way for users to book a ride. The website is designed to be user-friendly and accessible.

## Features
- Contact information for booking taxis.
- Clear description of the services provided.
- A call-to-action button that allows users to quickly call **1420** for booking a taxi.
- Simple and responsive design for easy access on both desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3
